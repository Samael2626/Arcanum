# Proyecto Arcanum

Este es el proyecto de desarrollo para la aplicación ARCANUM, una app de astrología y esoterismo.

## Semanas de Desarrollo

- [Semana 1: Backend FastAPI](./Semana-1-Backend-FastAPI.md) - Estructura, modelos DB, auth JWT + refresh tokens
- Semana 2: Motor astral
- Semana 3: Flutter: tema Grimorio Vivo + GoRouter + pantallas skeleton
- Y así sucesivamente según el plan establecido en el documento de estado.

## Estado Actual

Según el documento `ARCANUM-Estado-Sesion.md`, el proyecto se encuentra en estado "diseño-aprobado" y listo para arrancar la Semana 1.

## Tecnologías

- Backend: FastAPI (Python 3.12)
- Base de datos: PostgreSQL
- Caché: Redis
- Frontend: Flutter
- IA: Claude API
- Pagos: RevenueCat
- Deploy: Railway/Render + Supabase + Upstash
